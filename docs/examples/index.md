## Genel Kalıp

Çoğu senaryo şu kalıbı izler:

```
1. Konu aç
2. 5-10 tur otomatik tartışma
3. Müdahale et (send veya @)
4. 5-10 tur daha
5. /summary ile özet al
6. /export md ile raporu kaydet
```

Süre: 15-30 dakika. Token maliyeti (Mistral small): ~$0.01.
